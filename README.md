# 猜數字遊戲 (Guess the Number)

一個基於 HTML、CSS 與 JavaScript 網頁技術開發的單機版猜數字經典小遊戲。玩家需要在 1 到 100 的範圍內猜測系統隨機生成的秘密數字，系統會根據玩家輸入的數值給予對應的提示。

## 快速開始

本專案為純前端網頁應用，無需安裝任何複雜的相依套件或進行編譯，即可在瀏覽器中直接運行。

### 方式一：直接雙擊運行
1. 下載或複製本專案的程式碼。
2. 在檔案瀏覽器中找到 [index.html](file:///d:/Code/Guess-the-number/index.html)。
3. 按兩下 [index.html](file:///d:/Code/Guess-the-number/index.html)，即可使用預設瀏覽器開啟並開始遊玩。

### 方式二：使用本地開發伺服器運行
若您希望透過本地網頁伺服器（Local Server）開啟，可以使用 Bun 來啟動靜態伺服器：

1. 確保您的系統已安裝 Bun。
2. 在專案根目錄下執行以下指令安裝並開啟靜態伺服器：
   ```bash
   bunx serve .
   ```
3. 開啟瀏覽器並造訪 `http://localhost:3000`（或終端機輸出的對應埠號）。

## 遊戲功能

- **隨機生成密碼**：每次載入頁面時，系統會隨機產生一個介於 1 至 100 之間的整數作為秘密數字。
- **即時回饋提示**：
  - 當玩家猜測的數字大於秘密數字時，顯示紅色提示文字「Too high!」。
  - 當玩家猜測的數字小於秘密數字時，顯示橘色提示文字「Too low!」。
  - 當玩家成功猜中數字時，顯示綠色提示文字「Congratulations! You guessed it!」。
- **防呆輸入驗證**：若玩家輸入非數字、小於 1 或大於 100 的數值，系統會提示請輸入 1 到 100 之間的數字，避免玩家浪費猜測機會。

## 開發與自訂配置

### 遊戲邏輯修改
若您希望自訂遊戲的規則（例如修改數字範圍），可直接編輯 [index.html](file:///d:/Code/Guess-the-number/index.html) 的內容：

- **修改數字上限**：
  - 將核心腳本中生成隨機數的程式碼 `Math.floor(Math.random() * 100) + 1`，以及驗證邏輯中的最大值 `100`，修改為您想要的上限數值。
  - 同步更新 HTML 提示文字與 `<input>` 的 `max` 屬性。

| 自訂項目 | 相關程式碼位置 | 預設值 |
|----------|----------------|--------|
| 數字下限 | `min="1"`, `guess < 1`, `+ 1` | 1 |
| 數字上限 | `max="100"`, `guess > 100`, `* 100` | 100 |
